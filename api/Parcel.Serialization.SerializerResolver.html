    Class SerializerResolver

    Stored a collection of Serializers and resolves which Serializers should be used for each Type.

    Remarks

    Serializers that are registered with the SerializerResolver class will be stored. When an unseen type attempts to serialize, the SerializerResolver will check each registered Serializer and find the first one to return true from the CanSerialize(Type) call. This Serializer will be cloned and the appropriate ObjectCache instance will be placed into the Serializer.
